# Landing Page Playbook

Use the public side of the app as a structured publishing surface.

## Core page types

- Hero landing page for the main offer
- Feature pages for offline play, moderator flow, and setup speed
- Comparison pages versus paper rules or generic score trackers
- Evergreen guides that answer practical hosting questions

## Content workflow

- Add a markdown file
- Register it in the content catalog
- Publish it through the SSR route

That keeps the system simple while still giving you room to grow into a larger CMS later.
