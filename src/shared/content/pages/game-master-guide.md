# Game Master Guide

The public site can host long-form text pages that explain the game, teach facilitation, and attract search traffic.

## Good landing pages for a Mafia product

- How to host a Mafia night for 8 players
- Best roles for a beginner-friendly table
- Printable moderator checklist
- Party game rules for schools, retreats, and team events

## SEO-friendly structure

- One intent per page
- Strong title and introduction
- Clear calls to action into the game app
- Supporting internal links to adjacent guides

## SSR over SPA for these pages

SSR is the right default for text-heavy public pages because it improves first render, indexing, and share previews while keeping the interactive app separate.
